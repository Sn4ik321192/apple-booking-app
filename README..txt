# Telegram Booking Mini App

Telegram Mini App для онлайн-записи клиентов.

## Функции

- Запись клиента
- Выбор услуги
- Календарь
- Свободные и занятые слоты
- Профиль клиента
- Админ-панель
- Уведомления в Telegram
- Supabase база данных

## Установка

1. Создайте Telegram бота через @BotFather.
2. Создайте Supabase проект.
3. Импортируйте database.sql.
4. В script.js вставьте:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
   - BOT_TOKEN
   - ADMIN_CHAT_ID
   - ADMIN_ID
5. Загрузите файлы на Vercel, Netlify или обычный хостинг.
6. В BotFather укажите ссылку Mini App.
